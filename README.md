# WeenggsPractical

# React Dashboard (Vite + React + Tailwind + MUI)

## Features
- Login (dummy)
- Dashboard with stats (users from jsonplaceholder)
- User management (search, sorting, inline edit, drawer edit, delete with confirmation, pagination)
- Project management (create/edit/delete, inline & drawer)
- Tailwind CSS for styling, MUI icons for UI pieces
- Dark mode support (prefers 'class' toggling - use the Topbar button to toggle)

## Get started

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build:
```bash
npm run build
```

## Notes
- Default demo login: **admin / 123**
- Projects persist to `localStorage`
- Users are fetched from `https://jsonplaceholder.typicode.com/users`
